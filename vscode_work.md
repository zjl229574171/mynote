### 2.1. 核心语法速查表
  
| 功能          | 语法示例                      | 效果预览             |
|---------------|-------------------------------|---------------------|
| **标题**      | `# 一级` `## 二级` ...        | <h4>二级标题</h4>   |

| *斜体*        | `*文本*` 或 `_文本_`          | *斜体文字*          |
| ~~删除线~~     | `~~文本~~`                    | ~~删除文字~~        |
| 代码块        | ```` ```语言 [代码] ``` ````  | 带语法高亮的代码区  |

| 引用块        | `> 引用内容`                  | > 名言引用          |

| 有序列表      | `1. 第一项`                   | 1. 带序号列表       |
| [链接](/)      | `[描述](URL "提示")`          | [示例链接](#)       |
